How It Works
1. Data + Indicators
Fetches 3 months of OHLCV data via Alpaca, then engineers 13 indicators: RSI, MACD, Bollinger Bands, ATR, OBV, CCI, SMA, and more.
2. RL Training (PPO)
A PPO agent trades a simulated portfolio over historical data, learning which indicator combinations lead to profitable trades.
3. Live Prediction
The trained agent observes today's market state and outputs BUY / SELL / HOLD with a confidence score. Models are cached daily.
Note: If stable-baselines3 is not installed, the engine falls back to a rule-based indicator policy. Install with: pip install stable-baselines3 gymnasium
